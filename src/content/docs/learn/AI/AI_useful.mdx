title: AI 应用精选
description: 精选日常 AI 辅助工具
---

AI 技术日新月异，各类好用的 AI 工具层出不穷。
本篇收集的都是在日常中实际体验过、确实好用的 AI 工具。
那些仅听说过、未亲自使用，或者由于各种原因体验不佳的工具，都不会出现在此篇中。
随着技术的发展，若某些工具逐渐跟不上时代，它们也会从本篇中移除。
也因此，本篇具有强烈主观性质，真实情况请自行评判。

- [ChatGPT](https://chatgpt.com/) 无须多言
- [Deepseek](https://api-docs.deepseek.com/zh-cn/) 日常服务器不稳定，但是 API 价格在主流模型中最便宜，在自娱自乐场景中优先
- [秘塔](https://metaso.cn/) 能够指定数据库和真实链接实际资源内容的功能让它脱颖而出
- [Cursor](https://cursor.com/agents) 能够直接在 IDE 中操作
- [Warp terminal](https://www.warp.dev/terminal) 在终端中提供 AI 辅助，能够生成、优化和解释命令
- [kimi copilot - 网页总结助手](https://chromewebstore.google.com/detail/kimi-copilot-%E7%BD%91%E9%A1%B5%E6%80%BB%E7%BB%93%E5%8A%A9%E6%89%8B/icmdpfpmbfijfllafmfogmdabhijlehn?hl=zh-CN&utm_source=ext_sidebar) 第三方爱好者开发的可以总结网页内容的浏览器插件
- [BiliScope](https://chromewebstore.google.com/detail/biliscope-bilibili%E6%8F%92%E4%BB%B6%EF%BC%8C%E4%BD%A0%E7%9A%84b%E7%AB%99/ekmbchepcdggpcbdpjpijphjiiiimfga?hl=zh-CN&utm_source=ext_sidebar) 由 [码农高天](https://www.bilibili.com/video/BV1BX4y1U7uZ/?share_source=copy_web&vd_source=4e6e3d05c54fbafa49b2a00b5cc81815) 开发的B站视频总结插件
- [DeepWiki](https://deepwiki.com/) 能生成 Github 项目文档
