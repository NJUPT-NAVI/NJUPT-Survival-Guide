## 为什么需要它？
`南邮小程序`的微信小程序。但是他存在下面缺点:
1. 强联网依赖。进入小程序都需要连接互联网，在网络不好的时候无法使用。
2. 冷启动慢。小程序冷启动时间较长。
3. 无法和系统其他功能联动，例如：闹钟、AI、待办。

为了解决以上问题，我们需要一个专门的**课表**APP来解决这个问题。本文主要介绍**小爱课表**，如果你是其他的系统，例如**鸿蒙**，可以参考[用 kwgt 固定课表](./kwgt)

<Steps>

1. 初始化
   - 启用超级小爱
   - 输入 `课表` 进入课表界面 <img src={img2.src} alt="小爱课表示例 2" style="width: clamp(120px, 30vw, 250px); height: auto; border-radius: 8px;" />

2. **进入导入课表页面**
   - 在设置设置中点击导入课表，从教务系统导入<img src={img5.src} alt="小爱课表示例 5" style="width: clamp(100px, 20vw, 250px); height: auto; border-radius: 8px;" />
   :::caution[导入功能下线]
   如果您的版本提示这个公共，请参考[小米课程表恢复](https://www.bilibili.com/video/BV1zkaEzEEK5/)输入神秘代码来解决
   `window.location.hash ='/import'`
   :::
    
3. **进入教务系统**
   - 根据文本提示进入导入**个人课表页面**
   :::danger[进入正确的页面！]
    - 教务系统的初始密码是**完整身份证号**
    - 请进入**个人课表页面**之后才执行导入
   ::: 

4. **后续设置**
   - 根据提示，选择正确的周数和学期
   - 前往设置添加小组件到桌面
   - **Enjoy it!**

</Steps>
