## 晨跑要求

<CardGrid>
    <Card title="基本信息">
    - **对象**：大一、大二学生
    - **合格次数**：30次 / 学期
    - **时间**：周一至周五 `6:35` - `7:20`
    - **周期**：第二周 至 第十六周
  </Card>
    <Card title="路线与时长">
    - **路线**：图书馆 ↔ 新材料学科楼 (双向)
    - **距离**：约 1000 米
    - **起点打卡**：`6:35` - `7:10`
    - **终点打卡**：`6:40` - `7:20`
    - **时长限制**：3 ≤ 男生 ≤ 11分钟，3 ≤ 女生 ≤ 13分钟
  </Card>
</CardGrid>



## 晨跑奖惩

<CardGrid>
    <Card title="🏆 奖励与加分" icon="approve-check-circle">
    学期内完成 **30次** 以上可获体育课或体测成绩加分：
    - **35次**：+2分
    - **40次**：+3分
    - **45次**：+4分
    - **50次**：+5分
    
    :::tip
    参与校级体育活动、担任志愿者等也可获得次数奖励。
    :::
  </Card>
    <Card title="⚠️ 处罚与违规" icon="warning">
    - **次数不合格**：当学期体育课成绩按 **59分** 计。
    - **违规行为**：如穿拖鞋、代替打卡等，首次将通报并扣除 **5次** 有效次数；第二次则当学期晨跑成绩不合格。
  </Card>
</CardGrid>

## 其他事项

:::caution[注意事项]
- **有序进行**：不得穿拖鞋、不得使用自行车等代步工具。
- **刷脸优先**：优先使用摄像头刷脸（有企业微信实时提醒）。若无法识别，请及时更新照片。
- **预留时间**：建议提前安排出发，避免因排队拥堵导致超时，超时记录无效。
:::

:::note[特殊情况]
- **伤病减免**：因伤病无法参加，需提供医院证明等材料至 **训练馆102办公室** 申请减免。
- **天气影响**：遇雨雪、雾霾等天气将暂停晨跑，届时会发布通知。
- **成绩恢复**：大三、大四期间可补齐所欠次数，补齐后可恢复原始体育成绩。
:::

:::tip[信息查询与反馈]
- **次数查询**：南邮小程序 → 早锻炼模块
- **打卡时间**：企业微信 → 工具台 → 早锻炼人脸识别
- **问题咨询**：加入晨跑问题咨询 QQ 群 `1057982250`
:::
