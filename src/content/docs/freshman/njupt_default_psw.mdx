## 快速总览

| 系统 | 登录账号 | 初始/默认密码 | 备注 |
| ---- | -------- | -------------  | ---- |
| 邮箱系统 | 小写学号+\@njupt.edu.cn | 无(首次自行设置)  | 需先从“智慧校园 → 邮件系统”进入初始化 |
| 智慧校园 / 统一身份认证 | 学号 | 身份证后六位（X 大写）  |  |
| 南邮小程序 | 学号 | 同智慧校园  | 服务器偶有不可用情况 |
| 教务系统 | 学号 | 身份证号全部位数  | 与智慧校园密码不同，可单独修改 |
| 雨课堂（长江雨课堂） | 学号 | NJUPT + 学号后 6 位  | 需先绑定学校身份 |
| 财务处校园卡圈存 | 无 | 身份证后六位  | 充电费需要用到 |

## 使用建议

<Steps>
1. 统一身份认证先改强密码
   - 改完后再去初始化：邮箱、小程序、雨课堂等
2. 教务系统密码单独保留
   - 与统一认证不同，勿混淆
3. 启用多处记录
   - 本地密码管理器 / 纸质备忘
4. 遇到无法访问
   - 先判断系统是否宕机（群聊/通知）再考虑“忘记密码”
</Steps>

## 账号详解

<Tabs>
  <TabItem label="邮箱">
  
  **不存在“默认密码”**  
  首次需通过【智慧校园 → 邮件系统】入口初始化，页面会提示创建（设置）密码，设置完成后邮箱才真正开通。  

  **常见问题**
  - 无法进入：可能尚未为新生批量开通，稍等或看学院通知  
  - 登录用途：企业微信等  
  - 建议密码：与统一认证不同、使用随机强密码  

  **入口示意**（智慧校园内位置）：
  ![邮件系统入口示意](@assets/docs/freshman/njupt_default_psw/邮件系统在智慧校园的位置.webp)

  </TabItem>

  <TabItem label="智慧校园/统一认证">

  - 账号：学号 / 工号  
  - 初始密码：身份证后六位（如有 X 用大写 X）  （一般在入学时候已经重新修改过）
  - 修改方式：登录后“个人中心 → 修改密码”  
  - 作用：绝大多数系统的登录方式 

  <Aside type="caution">
  修改一次后请记录；统一认证被锁定可能需辅导员或信息化部门协助。
  </Aside>

  </TabItem>

  <TabItem label="小程序">
  
  - 南邮小程序使用与智慧校园同步的统一身份认证密码  

  <Aside type="tip">
  有时候小程序会不可用，不一定是密码错误，可能是服务器问题，也可以试试看重新加载。
  </Aside>

  </TabItem>

  <TabItem label="教务系统">
  
  - 网址：http://jwxt.njupt.edu.cn/  
  - 账号：学号  
  - 默认密码：身份证号全部位数（与统一认证不同）  
  - 可在首次登录后按提示修改为自定义密码  

  **重要提醒**
  - 教务系统与统一认证密码互不影响  
  - 忘记教务密码：联系辅导员协助重置  
    <Aside type="tip">
    稳定访问校园网放大校园网下直连：http://202.119.225.134/
    </Aside>

  </TabItem>

  <TabItem label="雨课堂">
  
  仅首次绑定学校身份时候需要使用，绑定后无需再次使用。

  默认密码：`NJUPT` + 学号后 6 位（示例：学号 B23020199 → NJUPT020199）  

  </TabItem>

  <TabItem label="校园卡圈存">

  - 密码：身份证后六位  
  - 圈存用途：校园网、充电费等  

  </TabItem>


</Tabs>

## 常见误区

<Steps>
1. 认为“教务系统密码 与 智慧校园密码” 相同
   - 实际：两套独立
2. 邮箱进不去就“密码错误”
   - 实际：可能尚未初始化
3. 小程序多次尝试仍无法登录认为“密码错误”
   - 实际：有时是服务端故障
4. 忘了是否改过某密码
   - 建议：启用本地密码管理器 / 纸质备忘
</Steps>
