## 我应该转专业吗？

转专业有利有弊，在做决定前，请仔细权衡以下几点：

<CardGrid>
    <Card title="转专业的优势" icon="approve-check-circle">
    - **追求热爱**：可以进入你真正喜欢或学校的强势专业。
    - **跨学科发展**：实现文理科互转，可能影响未来的职业方向和学位。
    - **更换环境**：若目标专业在另一校区，可以体验不同的校园生活。
  </Card>
    <Card title="需要考虑的挑战" icon="warning">
    - **补修课程**：需要补修新专业的部分课程，课程量取决于新旧专业的培养方案差异。
    - **影响绩点**：准备转专业的过程和后续的补课可能会占用大量时间，从而影响整体绩点。
    - **繁琐流程**：从申请到最终转入，涉及大量跨部门的交接手续。
    - **更换宿舍**：跨校区转专业必须更换宿舍，同校区也可能需要更换。
  </Card>
</CardGrid>

## 常见问题解答

### 我该转什么专业？

这完全取决于你的个人目标：
1.  **目标明确**：如果你已心仪某个强势专业，那就朝着这个目标努力。
2.  **目标计算机（不保研）**：如果想从事软件开发，且不打算保研，那么计、软、网院的差异不大。建议选择竞争压力相对较小的专业，以留出更多时间自学。
3.  **目标硬件**：可以考虑本校与硬件相关的对口专业。
4.  **考虑难度**：不同专业接收的**转专业名额**和考核难度各不相同，需要提前了解。

### 转专业需要什么条件？

通常大一或大二的学生有资格申请。

:::note[官方通知]
详细的转专业资格和最新政策，请务必查阅学工处网站的官方通知，例如：[本科生转专业管理办法（修订）](https://xsc.njupt.edu.cn/2024/0301/c17376a256014/page.htm)
:::

申请流程一般在每学年下学期启动，通过考核后，会在暑假放假前公布结果。考核通常包括两个核心部分：
1.  **绩点排名**：申请前所有学期的绩点排名需达到所在专业**前 15%**。
2.  **转专业考核**：由目标学院组织的**笔试**和**面试**。

:::tip[关键点]
很多人认为转专业考核最难，但实际上，对于**非强势专业**而言，**绩点排名前 15%** 才是最大的门槛，因为原学院需要根据这个标准来决定是否“放人”，尤其是在人数较少的专业。
:::

### 转专业考核考什么？

考核内容因学院而异，以下为部分参考：

<CardGrid>
    <Card title="计算机类学院">
    - **笔试**：大一转考 C 语言；大二转考 C 语言 + 数据结构。
    - **面试**：可能包含英文自我介绍、项目经历阐述，以及难度较高的 C 语言上机 OJ 算法题（但据说占比较低）。
  </Card>
    <Card title="其他专业">
    - 硬件相关的专业：考核内容通常围绕专业基础课，如**电路分析**、**高等数学**，部分专业可能还会考察**英语**。
  </Card>
</CardGrid>

:::caution
再次强调，具体的考核科目和要求，请以目标学院发布的官方文件为准。
:::

### 挂科会影响转专业吗？

**会。** 官方文件明确要求，申请转专业者**不能有挂科记录**。

### 转专业后，原专业的课程怎么办？

已修课程会根据情况进行处理：
1.  **新专业完全没有的课程**：可以选择保留绩点高的课程，或申请将绩点低的课程降级为任选课。
2.  **新专业有类似课程**：可以申请课程替代，通常要求更高难度的课程替代低难度课程（如高数 A 替代高数 B），学分会相应调整。
3.  **新旧专业完全相同的课程**：课程代码完全一致的课程会自动保留，无需处理。

### 转专业后如何补修课程？

你需要自行补修所有新专业有、而原专业没有的课程。
- **选课方式**：在每学期初或期末的选课阶段，自行在教务系统中选课。补修不额外收费。
- **时间冲突**：补修课程可能会与你的现有课程时间冲突，需要你合理规划。
- **关于降级**：大二转专业的同学，根据所缺课程数量，可能需要降级到下一年级，具体请遵循官方安排。

### 还有哪些注意事项？

<Steps>
1.  **及时交接**
    转专业成功后，请主动联系新旧两个班级的班委或辅导员，完成团组织关系、各类档案和文件的转接。

2.  **融入新集体**
    需要主动去认识新同学、融入新集体，这可能会影响一些需要投票或选举的活动。

3.  **生活细节**
    - 不同校区的晨跑规则、生活设施有所不同，需要重新适应。
    - 学生证也需要重新办理。

4.  **搬迁宿舍**
    跨校区转专业的同学，通常需要在开学前一周左右，按照学校安排从旧宿舍搬到新宿舍。
</Steps>
