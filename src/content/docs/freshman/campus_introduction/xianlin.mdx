### 食堂介绍

<Steps>

1. 南一
   - 定位：最早的三层老食堂，梅兰一条街末端，靠近**教学楼**（附近：**蜜雪冰城、二手书店**）
   - 楼层：
     - 1F 甜品/饮料 + 快餐；
     - 2F（外侧）新疆风味；
     - 3F 台球馆 / 家常菜 / 火锅
   - 高峰：工作日 11:40–12:10，18:00 前后
    :::tip[小贴士]
    - 3F 桌子上有免费插座
    :::

2. 南二
   - 定位：三层，靠近图书馆与晨跑起点
   - 楼层：
     - 1F（兰苑侧）**小卖部** / （菊苑侧）**烘焙店**；
     - 2F 称重自选菜；
     - 3F 健身器材 / 招聘活动
   - 高峰：晨跑后、11:45、18:00

3. 南三
   - 定位：最新翻新（2025 暑期）两层，含 **蜜雪冰城 / 麦当劳 / 小吃街**；外有**大型超市**
   - 楼层：
     - 1F 快餐 + 烘焙 + 品牌档口；
     - 2F 甜品店 + 咖啡店 
     - 2F 侧面 小卖部 + **麦当劳** + 小吃街延伸档
   - 高峰：晨跑日 7:20–7:40；午 11:50；晚 18:10 左右

</Steps>

### 快递站
> **详细内容请参考**[仙林校区快递站点信息](../../../../life/express/express_info_xl)
 - 地址:
   - **科创基地1号菜鸟驿站**：在东门内邮局左侧
   - **2号菜鸟驿站**：在东门内右侧，美团外卖柜右侧
     :::tip[小贴士]
     - 快递站营业结束：20:00
     - 菜鸟驿站快递首个序列数小于`30`的通常在**2店**，大于`30`的通常在**科创基地1店**
     :::
   - **顺丰、京东快递**：在1号菜鸟驿站左侧，李苑后门
