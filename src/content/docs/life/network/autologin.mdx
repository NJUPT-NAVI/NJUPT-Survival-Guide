## 为什么需要它？

正常登录需打开 <code>https://p.njupt.edu.cn/a79.htm</code>，手动输入账号密码。  
该工具将这一过程“本地构造 GET 包 + 直接发出”，实现**自动化**。

<Aside type="tip" title="原理简介">
校园网登录实质：向认证网关（示例：<code>https://p.njupt.edu.cn:802</code> 或其 IP）发送带用户名、密码的 GET 请求。  
脚本：跳过 DNS（直接使用 IP）+ 极小 C/汇编实现 → 减少解析与启动开销。
</Aside>

## 特性

<Tabs>
  <TabItem label="优点">
  1. <strong>极致速度</strong>：<code>version2_ASM.cpp</code> 平均 ~ <strong>8.430ms</strong>  
  2. <strong>梯子无影响</strong>：直连网关 IP，绕过慢速 DNS  
  3. <strong>零第三方依赖</strong>：仅标准 C + 少量汇编  
  4. <strong>Android 免编译</strong>：直接下载 Release 即用  
  5. <strong>可泛化</strong>：理论上支持结构相似的 GET 认证门户（改 URL 参数）
  </TabItem>

  <TabItem label="缺点">
  1. 需本地编译（除 Android）  
  2. 更换账号需改宏再重新编译（为速度进行硬编码）  
  3. 不提供桌面平台编译产物（安全 / 硬编码因素）  
  4. 网关 IP 若变化需重新修改源码  
  5. 无 SSL：流量明文（请避开不可信网络环境）
  </TabItem>

  <TabItem label="版本说明">
  <ul>
    <li><code>version1_API.cpp</code>：Win API 版本（略慢 ~1ms）</li>
    <li><code>version2_ASM.cpp</code>：汇编强化，速度最快</li>
    <li><code>version2_ASMcross.cpp</code>：跨平台 C/ASM 混合（Windows / Linux / macOS）<strong>推荐</strong></li>
    <li><code>Android v1.5</code>：APK 于 Release</li>
  </ul>
  <Aside type="note">
  默认在 <code>clang</code> 下测试。不保证 <code>msvc</code> 编译通过；若仅装 VS，请新建工程复制源码`version3_ASMall.cpp`编译。
  </Aside>
  </TabItem>
</Tabs>
