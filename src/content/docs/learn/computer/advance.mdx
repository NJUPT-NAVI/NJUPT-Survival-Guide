如果你不满足平常的使用, 希望探索一些更丰富的玩法

## Windows 系统

关于 Windows 系统, 可以看看[@冷雨的文章](https://ziheng5.github.io/2025/08/30/windows-configuration/), 可以让你在 Windows
上也舒适的使用命令行工具.

## Linux 系统

首先建议使用 KDE 桌面, 它的基础设施较为完整, electron应用剪切板和输入法这种问题在 KDE 出现的概率很低
(就笔者之前使用ArchLinux+KDE的经历而言, KDE 相当稳定)

关于 [Hyprland](https://hypr.land/), 可以看看 [这一篇](https://ziheng5.github.io/2025/08/17/hyprland-configuration/)

另外, 如果你想要使用平铺桌面管理器, 但是你的笔记本显示器太小, 可以考虑 [Niri](https://yalter.github.io/niri/)
