## 学生事务中心

:::caution[位置]

原地址：仙林校区图书馆一楼

现地址：南二食堂三楼
:::

> 服务大厅主要分为三个服务区和八个窗口，为同学们提供一站式服务。

## 各窗口职能

<CardGrid>
  <Card title="①② 财务窗口">
    - 校园卡充值
    - 财务缴费
  </Card>
  <Card title="③ 学生证/助理窗口">
    - 办理学生证业务（教程见[学生证补办](/life/aroundschool/service/card)）
    - 微信公众号后台答疑
  </Card>
  <Card title="④ 学工服务窗口">
    - 调入/调出通知单开具
    - 勤工助学岗位申请
    - 勤工助学筹金表收缴
    - 地方资助学校手续办理
    - 参军入伍政策咨询
    - 义务兵役学费减免材料办理
  </Card>
  <Card title="⑤ 教务处窗口">
    - 正方教务系统密码查询
    - 学科成绩及排名查询
    - 成绩单打印
    - 选课咨询
  </Card>
  <Card title="⑥ 信息办校园卡窗口">
    - 办理所有校园一卡通相关业务
  </Card>
  <Card title="⑦ 保卫处窗口">
    - 身份证明办理
    - 户籍关系咨询
    - 证件流程咨询与办理
  </Card>
  <Card title="学生助理区">
    - 为同学们提供贴心的引导和详细的解答。
  </Card>
</CardGrid>

## 线上及特色服务

<CardGrid>
    <Card title="线上平台">
        - **事务中心公众号**：定期发布与学习生活息息相关的热门资讯。
    - **失物招领平台**：丢失物品后可前往平台查询。
    </Card>
    <Card title="特色志愿服务">
        - “柚子小队”学业帮扶
    - 物资借用
    - 免费学习讲座
    - 免费证件照拍摄
    </Card>
</CardGrid>
