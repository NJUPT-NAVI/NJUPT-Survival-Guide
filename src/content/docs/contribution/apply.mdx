### GitHub Issue
本项目采用[GitHub Issue](https://github.com/NJUPT-NAVI/NJUPT-Survival-Guide/issues)的方式编写申请


#### 申请添加新功能
<Steps>
1. 进入[GitHub Issue](https://github.com/NJUPT-NAVI/NJUPT-Survival-Guide/issues)页面，点击`New issue`按钮。
2. 选择`新内容申请`模板。
3. 根据模板要求填写申请内容，尽可能详细地描述你希望添加的内容：
   - 请简要地描述您想增加的新内容
   - 您确认新增的内容符合[项目要求](https://njupt-navi.github.io/contribution/conventions/)吗
   - 该内容的具体描述
   - 该内容的参考资料
4. 提交申请，等待项目维护者的审核和反馈。
</Steps>

#### 申请删除有害内容
<Steps>
1. 进入[GitHub Issue](https://github.com/NJUPT-NAVI/NJUPT-Survival-Guide/issues)页面，点击`New issue`按钮。
2. 选择`内容删除`模板。
3. 根据模板要求填写申请内容，尽可能详细地描述您希望删除的内容
4. 提交申请，等待项目维护者的审核和反馈。
</Steps>
