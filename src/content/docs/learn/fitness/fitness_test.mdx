# 前言
首先你要知道，南邮大一到大四均有体测，且体育课测试和体测是不一样的，但是测一次可以既作为体育课成绩，也作为体测成绩。

这里将主要介绍南邮的体测规则。
# 体测规则
首先你要知道南邮的体测分数是怎么组成的，南邮的体测成绩是由各个项目加权计算得出（每个单项的满分都是100分）。

# 体测加分政策
体测是可以加分的，总共有三种方式：

1、跑操加分：跑操次数达到35次但低于40次、高于40次但低于45次、高于45次但低于50次、50次及以上，分别加2分、3分、4分、5分。

2、体测项目加分：超额完成体测项目可以加分，具体规则如图：
![体测项目加分](@assets/docs/learn/fitness/加分标准1.png)
![体测项目加分](@assets/docs/learn/fitness/加分标准2.png)


3、体育竞赛加分：参加省级及以上的体育竞赛可以加分，具体可以询问体育老师

# 体测注意事项
1、体测总共分为四个等级：90.0分及以上为优秀，80.0-89.9分为良好，60.0-79.9分为及格，59.9分及以下为不及格。
2、体测达到良好及以上才可以参与评奖评优，体测成绩不及格者在本年度可以补测一次，若仍不及格则记为不及格，若在毕业时体测成绩不达50分，则按结业或肄业处理。
3、跑操次数达到35次但低于40次、高于40次但低于45次、高于45次但低于50次、50次及以上，分别加2分、3分、4分、5分。
