## 三、交通路线
> [校车出发时间地点](/life/transportation/shuttle)

### 三牌楼校区

<Steps>

1. 民航
   - 南京禄口机场 → 民航交通车 → 瑞金路 → 民航售票处
   - 转乘 4 路公交 → 新街口
   - 换乘 34 路公交 → 三牌楼站 下车

2. 火车
   - 南京站 → 73 路 → 邮电大学站 下车
   - 或 13 路 → 三牌楼站 下车
   - 或 32 路 → 司背后站 下车
   - 或 45 路 → 东柏果园站 下车

3. 汽车
   - 汽车东站 → 45 路 → 东柏果园站 下车
   - 中央门汽车站 → 13 路 → 三牌楼站 下车
   - 或 45 路 → 东柏果园站 下车
   - 或 32 路 → 司背后站 下车

</Steps>

附近公交：

| 三牌楼站   | 13、42、32、78、100、16、31、34、152、816                             |

| 东柏果园站 | 95、318、45                                                          |
