---
title : 用 kwgt 固定课表
description : kwgt 本来是一个用于美化手机的工具，我用它来固定课表到屏幕上，方便查看
---

本来从手机上打开课表，需要进入南京邮电大学微信小程序，然后点击课表，等待加载资源返回。
有时因为网络原因，资源需要加载很久，而这样的操作还需要经历好多次。
于是可以把课表截图放到相册收藏，每次打开就直接看相册收藏。

但因为这个操作实在是太过频繁了，看到朋友有把课表作为锁屏的，但又显得难看。
去 github 翻看了一些导入课表的项目，但没有专门针对南邮的。
于是我转变思路，想有没有什么办法能把课表做成手机的服务卡片，直接贴在屏幕内，手机打开翻一下页面就能看到。
鉴于华为手机是没有自带这个功能的，于是可以使用本篇主题 `kwgt`。

`kwgt` 本来是一个用于美化手机的工具，可以自定义很多组件，让自己的手机变得非常酷炫，如果你想打发时间，不妨在B站上搜索相关内容。
但我仅仅只是需要用它来固定我的课表，就只用了很基础的功能（为满足固定课表的操作应均不用花费）。
